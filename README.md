# Recipe Management Application

This project is a full-stack web application that allows users to create, view, and manage cooking recipes in a single platform.

## Problem Statement
Many people store recipes across notebooks, screenshots, and multiple websites. This application provides a centralized, digital solution for organizing recipes efficiently.

## Core Features
- View a list of saved recipes
- Add new recipes with ingredients and instructions
- Edit and delete existing recipes
- AI-powered recipe suggestions and enhancements

## Technology Stack
- Frontend: React
- Backend: Node.js with Express
- Database: SQLite
- Containerization: Docker & Docker Compose
- Deployment: Cloud-based container hosting
- AI Integration: External AI API for recipe suggestions
